/backend/user/create (POST)
-arguments:
sessionid (session id of logged in medical staff/admin)
username (username of the user to be created)
password (password of the user to be created)
role (role of the user to be created)

-response:
HTTP response 201 if creation is successful
HTTP response 400 otherwise


/backend/user/login (POST)
-arguments:
username (username to login)
password (password to login)

-response:
sessionid(session id for authenticated user), HTTP response 200 if login is successful
HTTP response 400 otherwise


/backend/user/logout (POST)
-arguments:
-sessionid(session id for authenticated user)

-response:
-returns HTTP response 200 if successful
